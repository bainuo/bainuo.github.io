OS X 与 Linux 之间的高速数据交换！

Paragon ExtFS for Mac® 10 是一款低级文件系统驱动，专为解决 Linux 和 Mac® 之间的文件系统不兼容而开发。 ExtFS for Mac 实现 OS X下完全读/写访问 ExtFS 格式驱动。

基于Paragon UFSD™ （通用文件系统驱动）技术，此驱动能高速透明访问Ext2/3/4FS 分区，仿佛它们原本就是存在于 Mac 操作系统环境中一般，达到史无前例的高水平性能（类似于 HFS+）。 Mac OS X 程序能无限制访问这些 Ext2/3/4FS 分区，并且可以浏览内容，读且修改文件，创建新文件和文件夹，等等。

易于安装使用

通过用户友好的向导，该驱动非常容易安装。安装后，它将自动开机启动并且具有自动装载功能。使用 Windows 系统托盘将其唤起，安全退出！。

高性能

您可以在 Linux 格式分卷中存储并创建大于 4 GB 的文件，且支持完全读写访问，数据转移率与本地 Mac OS 扩展文件系统相似。

HFS+ 以及 ExtFS 高级支持

高级支持 HFS+ 以及 ExtFS 文件系统特征保证数据一致性（硬链接， 符号链接，等等
